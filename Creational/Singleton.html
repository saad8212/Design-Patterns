Singleton contains only one object which can not be modified, added or removed....it contains only a single object mostly it contains a configuration data of app, it is immutable and does'nt allow any other to modify it or recreate it

Example:

First create an IIF and assign it to a variable

var Singleton = (function(){ var config; function initializeConfiguration (data) { this.name = data.name; this.age = data.age } return { getName:function (data) { if(config === undefined){ config = new initializeConfiguration(data); } return config; } } })() now we will create instances, or modify but it will not allow var config1 = Singleton.getName({name:"abcd", age:123}); //initializeConfig {name: 'abcd', age: 123} var config2 = Singleton.getName({name:"abcdef", age:14}); //initializeConfig {name: 'abcd', age: 123}
